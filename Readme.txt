这个是我闲的没事写的一个小玩意儿，主要构想是以后连接上森亮号USB小灯的小游戏。
程序全英文因为懒得切换输入法状态，但是也没几句话。当它出现[!]的时候按下回车就是。
最后Quick Click会给出point和time。前者跟你的CPU有很大关系，所以你没法用这个和别人比较，这是自己参考用的；而后者是标准时间。point和time都反映了你的反应速度，前者较为精准，后者则是以秒做单位。
那么，你的反应速度有多快呢？
（与master的区别在于会调用Blink小灯。）
